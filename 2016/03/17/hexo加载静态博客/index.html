1.最基础的 方法 以及搭建blog需要下载的环境等问题 百度一大堆

2. 下面介绍一些简单必须的步骤

1.本地新建文件夹 eg：HJ’blog
2.进入该文件夹目录下 cd+HJ'blog
3.hexo init 随后会自动在目标文件夹中建立网站所需要的所有文件
4.现在我们已经搭建起本地的hexo 博客了 ，在浏览器localhost：4000 预览
hexo g  生产静态文件
hexo s    启动本地服务 ,样式查看
hexo d  上传

3.这里说一下跟换主题的坑

1.更换主题:妈了个鸡的  最大的坑 就是 配置主题 需要注意 空格和引号(重要的事情说3扁)  要特别 注意 千万别出错
2.hexo clean
3.git clone 你需要的主题
4.更改 主题的配置文件
