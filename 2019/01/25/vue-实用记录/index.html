1.动态路由匹配

官网链接

现在呢，像 /user/123 和 /user/456 都将映射到相同的路由。

一个“路径参数”使用冒号 : 标记。当匹配到一个路由时，参数值会被设置到 this.$route.params，可以在每个组件内使用。(let id = this.$route.params.id)

这样就不用在页面传参数的时候,额外加id这个参数key.

因为computed在生命周期beforeCreate()的时候就已经执行了.
